MFNotificationType

MFNotificationType enum specifies different type of storyline campaign.

  • MFNotificationTypeText: Notification type for showing alert/notification by text campaign.
  • MFNotificationTypeImage: Notification type for showing image campaign.
  • MFNotificationTypeVideo: Notification type for showing video campaign.
  • MFNotificationTypeUrl: Notification type for showing url webpage campaign.
