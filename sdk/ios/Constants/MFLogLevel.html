MFLogLevel

MFLogLevel enum specifies different levels of logging that could be used to limit or display more messages in logs.

Definition

typedef NS_ENUM(uint8_t, MFLogLevel ) {
   MFLogLevelNone = 0,
   MFLogLevelError = 1,
   MFLogLevelWarning = 2,
   MFLogLevelInfo = 3,
   MFLogLevelDebug = 4,
};
