MFBeaconEvent

MFBeaconEvent enum specifies different events of beacon.

  • MFBeaconEventEnter: Triggered event when user entering a beacon region.
  • MFBeaconEventExit: Triggered event when user exiting a beacon region.
  • MFBeaconEventImmediate: Triggered event when user range <1 metre from a beacon.
  • MFBeaconEventNear: Triggered event when user range is within 1-3 metres from a beacon.
  • MFBeaconEventFar: Triggered event when user range >3 metres from a beacon.
